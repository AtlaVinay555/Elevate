# Elevate

> A universal, system-level 'Scroll to Top' utility for Windows 11.

Whether you are deep in a 100-page PDF, endlessly scrolling a browser tab, or reading through an infinite chat interface, Zenith provides a unified, instant snap-back to the very top of your current view. 

## ✨ Features
* **Universal Compatibility:** Bypasses application-specific scrolling rules to work across standard Windows apps, browsers, and Electron wrappers.
* **Native OS Hook:** Designed to run efficiently at the system level with near-zero memory footprint.
* **Subtle UI Injection:** Configured to integrate cleanly into the Windows environment without clunky overlays.

## 🛠️ Tech Stack
* **Logic & Execution:** C++ / Windows API (Win32)
* **Rapid Prototyping:** AutoHotkey (AHK) scripts compiled to `.exe`
* **Customization:** Windhawk / Registry modifications for title bar injection
